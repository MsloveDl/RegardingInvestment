# 0. 题记
中国地市级政府（以下简称“市级政府”）在化解债务过程中，主要通过市场化机制间接利用A股市场进行债务消化。这一过程强调不新增隐性债务，并从隐性债务扩展到融资平台经营性债务（截至2024年末规模约14.8万亿元）。以下是具体方式，基于中央经济工作会议等政策框架，市级政府往往通过地方融资平台（LGFV，如城投公司）操作，这些平台债务占比大且复杂。

# 1. 市场化债转股（Debt-to-Equity Swaps）
 
* **机制**：市级政府指导下，将融资平台的经营性债务（如银行贷款或债券）转换为股权，引入战略投资者参与。这有助于改善平台的资产负债表，减轻利息负担，并通过股权形式盘活资金。债转股非强制，而是市场导向，适用于有实体业务和转型潜力的平台。债权人（如银行、资产管理公司）可转为股东，股权可能注入或关联A股上市公司，实现市场交易和增值。

* **与A股的关联**：如果平台或相关企业已在A股上市，或通过资产注入方式上市，债转股后股权可在A股流通，帮助消化债务。例如，市级政府作为股东，可注入优质资产（如基础设施）到上市公司，提升股价并通过再融资（如定向增发）筹集资金偿债。这间接利用A股流动性，避免直接债务违约。

* **市级应用**：市级平台债务多为经营性，债转股可减少财政兜底压力。政策鼓励金融机构（如银行、保险资金、养老金）参与，扩展到更多债转股项目。截至2025年，这一方式已成为化解经营性债务的核心路径之一。

## 案例

* **贵州茅台化债案例**：贵州省（省级，但机制适用于市级）通过茅台集团股权无偿划转至贵州省国资公司，随后在二级市场变现资金用于化解债务。2019年开始，该方式累计变现数百亿元，帮助缓解地方财政压力。（https://pdf.dfcfw.com/pdf/H3_AP202308181595068960_1.pdf）

* **安徽淮南市案例**：淮南市在2024年实现13家融资平台隐债清零，通过债转股等方式优化债务结构，引入战略投资者参与平台转型，减轻利息负担并盘活资产。（https://finance.sina.com.cn/stock/stockzmt/2025-04-01/doc-inerrtfn7399616.shtml）

* **重庆市级平台转型案例**：重庆部分市级融资平台通过债转股注入优质资产（如基础设施），转为股权形式，吸引银行和基金参与，化解经营性债务并提升市场融资能力。（https://36kr.com/p/3518431847209856）

# 2. 资产证券化（Asset Securitization）

* **机制**：市级政府将平台持有的稳定现金流资产（如供水、供热、停车场、保障性租赁住房）打包成资产支持证券（ABS）或房地产投资信托（REITs），通过市场发行筹资，用于置换或偿还隐性/经营性债务。这是一种市场化盘活存量资产的方式，减少对财政依赖。

* **与A股的关联**：ABS/REITs可在A股相关市场（如上海证券交易所）发行和交易，吸引投资者资金。市级政府通过此渠道实现债务优化，例如将基础设施资产证券化后，所得资金直接用于化债，间接提升相关上市公司的估值和融资能力。

* **市级应用**：适用于债务负担重的中西部或中小城市，政策督促市级主动化债，多措并举包括资产证券化。截至2024年，已有多个市级平台试点REITs发行，帮助消化数千亿元债务。 

## 案例

* **湖北省保障性租赁住房ABS案例**：湖北省发行首单保障性租赁住房ABS，将住房资产打包证券化，所得资金用于偿还地方政府债务，同时盘活存量资产，提升财政收入。（https://www.dahecube.com/article.html?artid=251531?recid=1）

* **福建省存量资产盘活案例**：福建省通过ABS和REITs将供水、供热等稳定现金流资产证券化，转化为现金流用于化解债务压力，同时引入混改合作，提升资源配置效率。（https://www.fxbaogao.com/detail/5180307）

* **全国首单“革命老区+长江经济带”CMBS案例**：湖北省发行该双贴标CMBS，将红色文旅资源证券化，变现资金直接用于地方化债，标志着文旅资产在化解债务中的创新应用。（https://www.dahecube.com/article.html?artid=251531?recid=1）

# 3.通过A股上市公司股权融资和质押

* **机制**：市级政府控股或参股的国有企业（SOE）在A股上市，通过IPO、定向增发（定增）或发行可转债等方式筹集资金，这些资金可回流用于偿还地方政府债务。同时，控股股东可进行股权质押，作为担保融资工具，缓解债务压力。

* **与A股的关联**：直接利用A股平台进行股权融资。地方政府债务扩张往往产生“挤出效应”，占用银行资源，导致上市公司控股股东增加股权质押比例（尤其是民营股东），以获得资金。这虽是间接结果，但有助于市级政府维持融资渠道。私有化或混合所有制改革后，股权可在A股交易，进一步消化债务。

* **市级应用**：许多市级SOE已在A股上市（如城投控股类公司），通过定增注入资产或融资，累计化解数万亿元债务。股权质押在高债地区更常见，异质性强：民营控股企业受挤出影响更大，而国有企业可能受益于政府倾斜。

## 案例
* **台州市利欧股份案例**：台州市政府提供纾困资金支持控股股东王相荣及其一致行动人，化解股票质押风险，通过股权融资方式注入资金，缓解上市公司债务压力。（https://www.stcn.com/article/detail/948162.html）

* **深圳市科陆电子案例**：深圳市国资委下属远致投资有限公司受让控股股东饶陆华部分质押股份，以10.34亿元资金帮助化解股权质押风险，并通过A股再融资维持流动性。（https://finance.qingdaonews.com/wap/2018-09/18/content_20213455.htm）

* **北京海淀区纾困基金案例**：北京市海淀区成立规模达百亿元的纾困基金，帮扶本地A股上市公司化解股权质押风险，通过国资入股和股权融资方式注入资金，避免平仓危机。（https://www.nbd.com.cn/articles/2018-10-18/1264216.html）

# 4. 总体政策框架与效果

* **政策支持**：中央通过“三支箭”（6万亿元专项债置换、4万亿元专项债化债、2万亿元特别国债支持）协同发力，市级政府需统筹房地产、地方债和中小金融机构风险。化债强调市场化、法治化，无新增隐性债，平均每年消化额显著降低。

* **效果与挑战**：这些方式有助于降低杠杆率（如从14.3万亿元隐性债降至2.3万亿元），但实际执行需防范道德风险。市级债务规模大（部分估算19万亿元，包括利息近1万亿元），A股利用虽有效，但依赖市场环境和投资者信心。

* **注意**：A股利用多为间接，非直接“股市化债”，而是嵌入市场化化解路径。实际案例因地区差异（如北京、上海已清零隐性债），需结合本地财政能力。
